== Description ==

Professional Slider for your Wordpress site.

= Features =

** 10 Unique Transitions ** ( fade out, fade in & other random )
** Support for multi-sliders. **
** Auto-Play. **
** Inline Content Support **
** Supporting all major web browsers including IE6 **
** Define specific transitions **
** Set the number of bars ** 
........................... and much more!!

== Installation ==

1. Extract and upload 'slider.zip' to your wp-plugins directory
1. Activate the plugin from the WP Panel.
1. From the admin panel click -> Slider to view or change settings
